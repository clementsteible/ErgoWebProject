A noter :

- Pour le journal, seule marche la rechercher pour toutes les émotions.
- Le compte administrateur est aussi un utilisateur enregistrer, c'est celui qui possède le plus de colonnes de Beck dans la BDD, et donc potentiellement le plus 
intéressant à utiliser pour faire des tests sur la recherche/statistiques.
- Les dates des évènements enregistrées dans la BDD pour classer les colonnes vont du 01/01/2018 au 05/01/2018. Mais n'hésitez pas à créer des colonnes vous même
pour tester tout cela en détail.

Pour lancer l'application : se placer dans le dossier "colonnes" et lancer un "python manage.py runserver". Puis taper "http://127.0.0.1:8000/" dans votre navigateur.

Notre application web est responsive, cela dit elle a été testé avec un émulateur de résolution de téléphone portable. 
Sous Chrome > Plus d'outils > Outils de développement > Toggle device toolbar. Puis sélectionner le Nexus 5X (résolution : 412 x 732) pour une navigation optimale.
